# DataXlabs_Task2
📌 Project Overview
This Business Intelligence project transforms raw, historical airline data sourced from Kaggle into an interactive analytical ecosystem. The project uncovers hidden patterns in flight operations, identifies major bottlenecks causing delays, and evaluates the core drivers behind passenger satisfaction to improve airline efficiency.
🚀 Key Insights DeliveredOperational Bottlenecks: 
Pinpointed the primary root causes of flight delays (e.g., Late Aircraft vs. Weather).Customer Pain Points: Isolated the exact touchpoints (e.g., In-flight Wifi, Baggage Handling) that trigger poor passenger reviews.Route Profitability & Efficiency: Mapped high-frequency routes against average delay times to find operational inefficiencies.
